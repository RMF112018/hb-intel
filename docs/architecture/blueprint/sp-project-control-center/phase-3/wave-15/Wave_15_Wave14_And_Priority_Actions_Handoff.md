# 11 — Wave 14, Priority Actions, and Handoff

Wave 14 owns checkpoint queue, route step, decision, audit trail, and decision history. Wave 15 owns the underlying External Systems source/review records.

Handoff triggers include custom link approval, mapping correction, unresolved mapping conflict, blocked-by-policy exception, stale mapping blocking readiness, and launch issue affecting project readiness.

Priority Action candidate dedupe key: `projectId | systemKey | issueType | subjectKey`.
