# Wave 16 Storage Split And Canonical Schema References

## Storage Split Summary

- Global platform and policy defaults: HBCentral-backed canonical policy/definition contracts.
- Project effective settings: project-site effective value contracts.
- Project override workflow: project-site overrides and change-request contracts.
- Validation/audit/health evidence: project-site validation, audit-event, and health snapshot contracts.
- Secrets: approved secret stores only; SharePoint stores references only.
