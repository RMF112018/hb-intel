# Wave 14 and Wave 13G Estimating Checkpoint Contract

## Authority Split

- Wave 13G remains Estimating Workbench feature authority.
- Phase 14 / Wave 14 governs checkpoint queue, routing-step, decision, stale/supersession, and audit semantics for estimating-related approval prompts.

## Freeze/Handoff Semantics

Estimating freeze and handoff checkpoints are governed by Phase 14 queue/decision contracts while estimate domain behavior remains Wave 13G-owned.

## Prohibited Reassignment

This contract does not transfer Estimating Workbench feature ownership from Wave 13G and does not authorize runtime writeback.
