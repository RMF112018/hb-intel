# Wave 14 HBI Guardrails and Audit Model

## HBI Guardrail

HBI may summarize and cite evidence only. HBI has no decision authority, no checkpoint approval authority, and no override authority.

## Audit Model Anchor

Wave 14 governs audit semantics for checkpoint/decision history, preserving source lineage and permission constraints.

## Non-Authorization

No runtime AI-action execution, source-system writeback, or tenant mutation is authorized.
