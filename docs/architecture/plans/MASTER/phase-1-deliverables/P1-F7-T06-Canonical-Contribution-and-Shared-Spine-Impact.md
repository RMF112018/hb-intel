# P1-F7-T06: BambooHR Canonical Contribution and Shared Spine Impact

## Thin Canonical Core

- BambooHR contributes to the shared workforce identity spine, assignment/staffing spine, and person-to-project mapping spine.
- Canonical mapping must remain minimal and traceable, translating only the identities and staffing attributes required by downstream read models.

## Dependencies

- Durable identity mapping is required before BambooHR people records are used as a canonical cross-system reference.
