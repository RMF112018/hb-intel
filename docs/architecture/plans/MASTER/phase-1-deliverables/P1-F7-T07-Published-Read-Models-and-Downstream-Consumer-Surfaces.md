# P1-F7-T07: BambooHR Published Read Models and Downstream Consumer Surfaces

## Published Read Models

- Workforce identity directory publication
- Staffing and assignment publication aligned to project and team consumers
- Permission-aware workforce-change views for admin and operator surfaces

## Consumer Boundary

- Consumers use governed repositories and `@hbc/query-hooks`.
- No direct BambooHR access is allowed from PWA source modules or feature packages.
