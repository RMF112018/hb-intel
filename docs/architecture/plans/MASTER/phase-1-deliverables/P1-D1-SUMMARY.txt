  ✓ Write safety code belongs in @hbc/data-access (Layer 2)
  ✓ Session state / offline queue NOT duplicated
  ✓ Retry logic wraps HTTP client, not repositories
  ✓ Idempotency keys generated by frontend, sent as header
  ✓ Backend honors keys via Azure Table Storage
  ✓ Audit logging async-non-blocking
  ✓ Package boundaries maintained
