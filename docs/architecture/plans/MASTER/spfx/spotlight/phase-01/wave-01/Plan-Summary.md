# Plan Summary — Wave 01

Wave 01 is the architecture wave.

The current repo shape is healthy enough that the redesign should happen inside the shared surface and its immediate mapping seams, not as a broad homepage rewrite.

## Required outcomes
- an explicit `SpotlightLayoutMode` contract or equivalent
- a mode resolver driven by practical usable space
- featured essentials separated from optional details
- a keyboard/touch-safe details disclosure
- a keyboard/touch-safe past-spotlights/history disclosure
- a less monolithic surface tree that can be validated by mode

## Closure expectation
At the end of Wave 01, compact/minimal states must show materially less information by default.
