## Headline conclusion

The Spotlight has a strong architectural foundation but a materially underperforming hosted runtime. The biggest gap is the combination of unreliable / absent media and an oversized first-view media footprint, which causes the featured story to start too low and makes the module feel emptier than it should in the homepage’s dominant first-row slot.
