# Project Portfolio Spotlight — Wave 01 Prompt Package

## Wave objective

Close the highest-value doctrine and target-state gaps in the current Spotlight without disturbing the overall architecture that is already directionally correct.

## Scope posture

This wave is **not** a broad redesign.
This wave is a disciplined closure pass focused on:

- compact/minimal-state selectivity
- disclosure and icon compliance
- runtime failure-state clarity
- first-paint layout stability
- CTA and masthead governance

## In-scope outcomes

- smaller modes show less by default
- details/history disclosures remain explicit and credible
- no Unicode pseudo-icons remain
- runtime errors produce a professional Spotlight error state
- first render does not briefly pretend the surface is wide when it is not
- CTA hierarchy becomes cleaner

## Out of scope

- broad CSS token migration
- deep overlay-engine replacement
- list-schema redesign
- unrelated homepage-shell work
