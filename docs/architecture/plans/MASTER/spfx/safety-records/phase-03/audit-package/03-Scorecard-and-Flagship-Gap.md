## Hard-stop failures

1. **Generic prototype / enterprise-module dominant posture**
2. **Missing credible state-model implementation across the route set**
3. **Weak breakpoint and compact-state definition**
4. **Insufficient hosted/package proof**
5. **Nominal shell/list/dashboard usage without real authored layout contracts**
6. **Exposed incomplete module (`Incidents`) inside the primary workspace nav**

## Exact flagship gap to close

The biggest delta is not backend correctness.  
The biggest delta is **presentation architecture**:

- real workspace masthead
- real sectioned layout system
- real responsive behavior
- real state orchestration
- real safety-specific surface primitives
- real closure proof
