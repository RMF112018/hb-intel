# Benchmark Reference Summary

## Governing translation

The benchmark package does **not** require `PriorityActionsRail` or `PriorityActionsRailAdmin` to look like HB Kudos.

It does require:
- equivalent implementation rigor
- equivalent backend/data discipline
- equivalent UX completeness
- equivalent runtime maturity
- equivalent closure proof

### Public rail

- utility-first

### Admin
- authoring-oriented
- clear

- trustworthy
- preview-rich

- purpose-fit sophistication and persona expression

- state orchestration

- `PriorityActionsRail`: target `36+/40`
