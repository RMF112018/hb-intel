## Why this package is materially better

### Better decomposition
The enhanced package separates:
- correctness and persistence trust
- admin product workflow
- shared/public surface behavior
- contracts / validation / preview parity
- styling closure
- hosted proof / docs integrity

### Better sequencing
The original wave sequence assumed the earlier product model was already trustworthy. The new sequence no longer makes that false assumption.

### Better end-state definition
Each prompt defines:
- what is wrong now
- what the correct future state actually looks like
- which seams and files to inspect
- how closure must be proven

### Better protection against false closure
The enhanced package explicitly blocks:
- cosmetic-only fixes
- undocumented contract drift
- stale scorecard reuse
- proof claims without runtime evidence
