# Phase 05 Prompt Package — Navigation Governance and Branding Rules

This package is the next phase after Phase 04 closure.

## Why Phase 05 is next

Phase 04 closed Lane B (`apps/hb-shell-extension`) as a real shell-extension product lane with top and bottom placeholder capabilities, activation governance, safe failure posture, and documented boundaries. The Phase 04 completion notes explicitly defer **Lane C navigation governance** as the next major work area. Phase 05 therefore moves out of custom React surface implementation and into supported SharePoint governance for home site, global navigation, branding alignment, and page-template standards.

## Package objective

Establish the governance, documentation, and implementation rules for **Lane C — Navigation & Governance** so HB Central can scale without degrading the now-stabilized Lane A homepage product or the new Lane B shell-extension lane.

## Sequence

1. `Phase-05-01-Home-Site-and-Global-Nav-Governance.md`
2. `Phase-05-02-Branding-and-Page-Template-Rules.md`

## Package contents

- README
- implementation summary
- 2 numbered execution prompts
- risk exposure
- standards / best practices

## What this phase is in scope for

- SharePoint home-site and global navigation governance
- hub/home-site branding alignment
- navigation taxonomy and ownership rules
- homepage template and modern-page composition rules
- editorial and authoring guardrails that preserve the Phase 01–04 system
- documentation updates needed to make Lane C operational

## What this phase is not for

- New homepage webparts
- New shell-extension React features
- Unsupported SharePoint shell takeover
- Homepage property-pane implementation
- Homepage async data integration
- Cross-session alert persistence
- Backend-driven personalization or analytics automation

## Expected outputs from execution

- Canonical Lane C governance docs
- Home-site / global-nav governance model
- Branding and page-template rules
- Admin/operator guidance for supported SharePoint configuration
- Acceptance and verification evidence package for the governance layer
