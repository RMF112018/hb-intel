# Phase 01 — Signature Hero Hard-Reset Package

## Objective

This package instructs the local code agent to scrap the current flagship homepage hero implementation and rebuild it from zero as a premium full-width identity surface for SharePoint.

The rebuilt hero must include only:

1. Hedrick Brothers logo / brand lockup
2. Tagline: **Build with GRIT.**
3. Personalized welcome message

Everything else is removed from the flagship hero:
- no CTA
- no metadata
- no editorial copy
- no secondary blocks
- no adjacent welcome card
- no gradient wash
- no split top-band architecture

## Package Order

Run these prompts in sequence:

1. `Prompt-01-Forensic-Hero-Reset-and-Repo-Truth.md`
2. `Prompt-02-Rebuild-Flagship-Hero-From-Zero.md`
3. `Prompt-03-Manifest-Toolbox-And-Top-Band-Cleanup.md`
4. `Prompt-04-Validation-And-Sppkg-Proof.md`

## Hard Gates

- The hero must use the full available width of the SharePoint full-width section.
- The hero must be a single flagship surface.
- The hero must not rely on a gradient background.
- The hero must be rebuilt from scratch, not incrementally styled from the current version.
- Only the Signature Hero should remain visible in the SharePoint toolbox for this build cycle.
- Do not re-read files that are still within your current context window or memory. Re-open files only when necessary to verify changed or unresolved details.

## Expected Deliverables

- updated hero component implementation
- updated hero CSS module implementation
- manifest cleanup
- hidden-from-toolbox updates for non-hero webparts
- updated docs reflecting the reset
- clean `.sppkg` proof for the reduced rollout scope
- validation notes with SharePoint runtime evidence

## Background Art Direction

The new hero background must feel premium, quiet, and contemporary. Preferred order:

1. authored wide-format project/material photography with low clutter
2. dark matte material / graphite / charcoal textured surface
3. premium neutral fallback field with subtle grain and tonal depth

The hero must not use:
- blue/orange gradient washes
- obvious glow effects
- stock intranet banner styling
- loud decorative overlays
- giant logo-dominant composition

## Notes

This package is intentionally aggressive. It assumes the current flagship hero direction should be discarded rather than refined.
