## Post-Upload Verification Steps

After uploading `hb-webparts.sppkg` to the SharePoint App Catalog:

1. **Clear SharePoint cache**: Force-refresh the homepage (`Ctrl+Shift+R`) or use a private/incognito window
2. **Verify hero renders**: The homepage hero should show:
   - Hedrick Brothers logo + "HB Central" label (subtle, top)
   - "Build with GRIT." (dominant tagline)
   - Personalized greeting ("Good {morning/afternoon/evening}, {Name}.")
   - Dark charcoal background with grain texture
3. **Verify hero is full-width**: The hero should fill the SharePoint full-width section without narrow-rail bottlenecking
4. **Verify no gradient**: No blue/orange gradient wash — only charcoal field
5. **Verify toolbox**: Only "HB Signature Hero" should appear in the HB Intel toolbox group; all other webparts should be hidden
6. **Verify edit mode**: The hero should render correctly in SharePoint edit mode with no configuration errors
7. **If stale hero persists**: Check the App Catalog for version conflicts — retract the old package, re-upload, and re-deploy
