# Prompt 07 — Editorial Communications Redesign

## Objective

Rebuild the communications surfaces so they feel authored, branded, and premium instead of tidy content boxes.

## Primary Scope

- `apps/hb-webparts/src/webparts/companyPulse/`
- `apps/hb-webparts/src/webparts/leadershipMessage/`
- `apps/hb-webparts/src/webparts/peopleCulture/`
- relevant editorial / recognition shared primitives

## Hard Instructions

- Do not reread files already in current context or memory.
- The communications modules must not remain visually interchangeable.
- “Featured + secondary cards” is not enough unless it is materially re-authored.
- This phase must improve emotional tone, editorial authorship, and perceived maturity.

## Required Work

1. Rebuild Company Pulse so it feels like a premium editorial content surface with stronger feature treatment.
2. Rebuild Leadership Message so it has gravitas, clarity, and a more intentional authored voice.
3. Rebuild People and Culture so it has warmth, recognition energy, and visual distinction from the more formal editorial modules.
4. Improve imagery, attribution, metadata, CTA hierarchy, spacing, and component rhythm.
5. Create or upgrade any shared editorial and recognition primitives needed to support the redesign.
6. Ensure the communications row or zone has a coherent internal hierarchy and does not collapse into same-looking boxes.

## Required Validation

- Provide before/after proof.
- Explain how each module now differs in tone and role.
- Validate focus, motion, responsive behavior, and authoring fallbacks.

## Deliverables

- redesigned communications implementations
- shared primitive changes where needed
- closure note focused on editorial authorship and emotional differentiation
