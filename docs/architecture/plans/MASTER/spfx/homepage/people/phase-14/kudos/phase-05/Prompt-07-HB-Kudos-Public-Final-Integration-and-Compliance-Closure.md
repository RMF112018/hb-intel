# Prompt-07 — HB Kudos Public Final Integration and Compliance Closure

```md
Objective

Conduct the final integration and closure pass for the **public-facing HB Kudos UI remediation effort**.

Primary Intent

Validate and, where necessary, finish the public HB Kudos surface so it now clears the intended bar as a premium, presentation-lane, SharePoint-hosted recognition feature.

Critical Constraints

- Use live repo truth.
- Do not begin by re-reading files already in active context or memory.
- Keep the final pass disciplined and evidence-based.
- Maintain strict compliance with `@hbc/ui-kit`, homepage doctrine, and package-boundary expectations.
- Do not claim strict compliance where only partial closure exists.

Scope

This prompt should:
- integrate any remaining loose ends from Prompts 01–06
- validate the rendered public experience as a whole
- close small remaining UI defects where appropriate
- produce a final closure report against the remediation matrix

Required Evaluation Areas

Evaluate the final public HB Kudos UI for:
- composition and visual authority
- featured recognition quality
- archive/recent-recognition quality
- CTA hierarchy
- composer quality
- recipient-selection quality
- responsive and zoom resilience
- accessibility and interaction polish
- ui-kit/shared-surface governance compliance
- overall rendered premium quality

Deliverables

1. Implement any remaining closure work needed.
2. Produce a concise final change summary.
3. Provide a remediation-matrix closure report with each row marked as:
   - Closed
   - Partial
   - Not Closed
4. For every non-closed or partial row, explain why.
5. State plainly whether the public HB Kudos surface now clears the intended product bar.

Important Rules

- Be rigorous, not optimistic.
- Do not treat “looks better” as sufficient closure.
- Do not hide partials.
- If something is still weak, say so directly.

Acceptance Standard

This prompt is successful only if the agent can credibly demonstrate that the public HB Kudos UI has been brought materially closer to flagship-quality presentation-lane standards and can clearly account for any remaining gaps.
```
