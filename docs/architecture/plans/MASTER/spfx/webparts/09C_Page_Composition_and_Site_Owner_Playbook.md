# 09C — Page Composition and Site Owner Playbook

## Purpose

Provide Prompt-09 page-composition guidance so HB Central site owners can maintain a premium, scannable homepage layout under SharePoint constraints.

## Recommended Zone Order

1. Top Band (Welcome + Hero)
2. Utility Zone (Priority Actions + Tool Launcher)
3. Awareness Zone (Pulse + Leadership + People/Culture)
4. Operational Awareness Zone (Project Spotlight + Safety/Field Excellence)
5. Discovery Zone (Smart Search / Wayfinding)

## Above-the-Fold Guidance

- Keep Top Band and at least one Utility webpart above the fold.
- Reserve awareness/operational content for below-the-fold first scan pass.
- Avoid stacking multiple heavy editorial surfaces at the top of the page.

## Coexistence With Microsoft Webparts

- Microsoft webparts are allowed when they fill non-overlapping needs (for example: quick document rollups, calendar snapshots).
- Place custom webparts first for branded hierarchy and consistent authoring behavior.
- Avoid duplicate function surfaces (for example, custom discovery + generic search box redundancy in the same zone).

## Section Layout Guidance and Anti-Patterns

- Prefer two-column sections for paired surfaces (utility, awareness, operational awareness).
- Use single-column only when a zone requires full-width hierarchy clarity.
- Avoid:
  - three or more dense custom webparts in one horizontal row,
  - mixing unrelated zone purposes in the same section,
  - uncapped promoted lists or unmanaged CTA sprawl.
