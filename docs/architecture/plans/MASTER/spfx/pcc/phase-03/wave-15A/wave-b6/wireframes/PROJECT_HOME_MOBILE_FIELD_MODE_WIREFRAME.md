## Mobile acceptance

- User should not need to scroll through a 2,500px Priority Actions wall before seeing the rest of the command center.
- Priority rows should wrap cleanly.
- Touch targets must be safe.
- Critical information must not rely on hover.
- Secondary detail belongs below the command layer.
