## Design intent

- The first fold should answer "what do I need to know and act on today?"
- The core-control cluster should read as the PCC operating layer.
- Lifecycle and HBI should be visible as differentiators, not buried.
- Reference cards should support the page without competing with the command layer.

## Guardrails

- No nested cards.
- No hidden DOM walls.
- No live writeback or action execution.
- No source ownership ambiguity.
