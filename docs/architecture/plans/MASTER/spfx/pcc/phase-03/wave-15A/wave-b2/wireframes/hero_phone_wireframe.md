# Phone Hero Wireframe

## Target Phone Layout

```text
┌──────────────────────────────────────┐
│ PROJECT CONTROL CENTER                │
│ Project Control Center                │
│ Project Home                          │
│ [Active Construction]                 │
│                                      │
│ Location: Sample City, ST             │
│ [Project Details] [Search Preview]    │
└──────────────────────────────────────┘
                seam
┌ Project Home │ Team │ Documents │ ... →
```

## Phone Rules

- Always show primary title and active surface.
- Show project stage as compact chip.
- Location may remain visible if space permits.
- Estimated value and scheduled completion may collapse behind `Project Details`.
- Search preview may hide if it crowds identity.
- Tabs scroll horizontally and remain text-based.
- Do not introduce hamburger navigation in this remediation.
