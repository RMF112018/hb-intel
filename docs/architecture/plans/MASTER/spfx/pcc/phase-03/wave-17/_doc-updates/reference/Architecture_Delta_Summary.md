# Architecture Delta Summary

## What Wave 17 Adds

- Canonical Site Health architecture authority.
- Complete domain model and health-finding lifecycle.
- Health category/status/severity/action registries.
- Desired-state versus observed-state comparison rules.
- Drift classification model.
- Repair-readiness and admin-verification workflow contracts.
- GET-only read-model route map.
- SharePoint list schema and index strategy.
- Persona-based redaction and action matrix.
- SPFx UX and accessibility contract.
- Wireframe package.
- HBI evidence/citation/no-authority guardrails.
- Codex/local-agent execution discipline.

## What Wave 17 Does Not Add

- Runtime implementation.
- Tenant mutation.
- Automatic repair.
- Permission mutation.
- Settings writeback.
- External-system writeback.
- Production deployment.
- Legal, claim, accounting, entitlement, compensability, or delay-damages determinations.
