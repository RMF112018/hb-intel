# Wave 12 Constraints Log Scope Lock

## Official Module

**Constraints Log — Make-Ready Constraint & Risk Exposure Center**

## Final Scope Decision

Wave 12 owns make-ready constraint management and embedded risk/exposure assessment for project execution readiness.

## In Scope

- Constraint identification and assignment.
- Ball-in-court ownership.
- Need-by, promised, forecast, and actual resolution dates.
- Constraint exposure scoring.
- 5×5 risk matrix.
- Initial and residual risk scoring.
- Risk response planning.
- Risk-to-constraint conversion.
- Constraint-to-issue/delay/change linkage.
- Make-Ready Board.
- Constraint Exposure Matrix.
- Command Center.
- Weekly Huddle mode.
- Root Cause & Lessons Learned view.
- Priority Action candidate generation.
- Workbook source traceability.
- Documentation-only target architecture.

## Out of Scope

- Source/runtime code changes in this documentation wave.
- Formal CPM schedule editing.
- Primavera/P6 schedule mutation.
- Legal advice or contract interpretation.
- Claim entitlement or compensability determination.
- Delay-damages calculation.
- Formal forensic schedule analysis.
- Change-order approval execution.
- Permit application submission.
- Inspection scheduling.
- Document storage or evidence upload.
- SharePoint/Graph/Procore/Primavera/AHJ/Sage writeback.
- External-system scraping, polling, mirroring, or sync.
- Tenant mutation.
- Production rollout.
