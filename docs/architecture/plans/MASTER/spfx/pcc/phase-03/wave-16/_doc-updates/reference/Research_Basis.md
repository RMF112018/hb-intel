# Research Basis

## Microsoft / SharePoint / Graph
- SharePoint list-view threshold and indexed-filter discipline inform list schema decisions.
- Microsoft Graph throttling guidance informs caching, retry, and no-polling posture.
- Microsoft Graph change notifications inform future event-driven revalidation posture.
- SPFx client-side web part guidance informs client/read-only UI boundaries.
- Microsoft Graph least-privilege guidance informs Graph permission posture.
- Microsoft Purview audit activity guidance informs separation of PCC business audit from compliance/security audit.

## UX and approvals
- Admin/settings UX should be grouped, searchable, overview-first, and use detail drawers/dialogs carefully.
- Approval workflow references inform request/review/apply lifecycle and self-approval controls.
