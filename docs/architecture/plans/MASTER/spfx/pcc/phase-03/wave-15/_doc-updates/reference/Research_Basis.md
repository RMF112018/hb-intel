# Research Basis

- [SharePoint Framework overview](https://learn.microsoft.com/en-us/sharepoint/dev/spfx/sharepoint-framework-overview) — SPFx is recommended SharePoint customization model and runs in browser/current user context.
- [Microsoft Graph permissions overview](https://learn.microsoft.com/en-us/graph/permissions-overview) — Delegated vs application permissions and least privilege.
- [Microsoft Graph throttling guidance](https://learn.microsoft.com/en-us/graph/throttling) — Avoid continuous polling; honor Retry-After/backoff in future live-read integrations.
- [SharePoint list view threshold](https://learn.microsoft.com/en-us/sharepoint/troubleshoot/lists-and-libraries/items-exceeds-list-view-threshold) — 5000-item threshold and need for indexed query planning.
- [Azure Key Vault security features](https://learn.microsoft.com/en-us/azure/key-vault/general/security-features) — Soft delete, purge protection, rotation for future secret posture.
- [OWASP API Security Top 10 2023](https://owasp.org/API-Security/editions/2023/en/0x11-t10/) — BOLA, BFLA, SSRF, improper inventory, unsafe consumption of APIs.
- [EarthCam API / Broadway Media Player](https://www.earthcam.net/api/) — Camera embed/API capabilities exist but are future-gated.
- [OxBlue embed camera images](https://knowledge.oxblue.com/embed-camera-images) — Camera image embedding exists but is future-gated.
- [Sensera SiteCloud public access URLs](https://senserasystems.com/support-article/keep-your-stakeholders-in-the-know-with-public-access-urls/) — Public camera pages exist; PCC embed is TODO.
