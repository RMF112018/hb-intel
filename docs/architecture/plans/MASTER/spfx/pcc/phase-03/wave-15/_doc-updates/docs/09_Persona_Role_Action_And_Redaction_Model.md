# 09 — Persona, Role-Action, and Redaction Model

## New Personas

| Persona | Permission Scope | Description |
|---|---|---|
| `accounting-specialist` | per project | Assigned to specific projects; may view/edit accounting launch context for assigned projects. |
| `accounting-manager` | all projects | Standard all-project access for accounting launch context and mapping visibility. |

## Approved Link Editors

Project-level launcher links may be configured by Estimating Coordinator, Lead Estimator, Project Executive, Project Manager, IT/Admin, and Manager of Operational Excellence.

## Custom Link Approval

Custom links require review/approval by Project Manager or Project Executive.

## Redaction

External contributors and viewers receive redacted source-health/access issue details. Accounting links are visible to accounting personas, PM, PX, IT/Admin, and PCC Admin. HBI can cite/link only records visible to the requesting persona.
