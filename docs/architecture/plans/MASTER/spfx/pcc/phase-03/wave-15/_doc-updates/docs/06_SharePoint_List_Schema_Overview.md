# 06 — SharePoint List Schema Overview

## Existing Lists Used as Inputs

Wave 15 uses existing HB Central list schema truth for project identity, user visibility, global config precedent, and launcher precedent. It does not mutate existing lists in this documentation package.

## New Lists Required

### HB Central

1. `PCC External System Definitions`
2. `PCC External URL Policy Registry`

### Project Site

1. `PCC Project External Launch Links`
2. `PCC Project External System Mappings`
3. `PCC External Object References`
4. `PCC External Review Items`
5. `PCC External System Health Snapshots`
6. `PCC External System Audit Events`

Full schema details are in `docs/sharepoint-schemas/` and `artifacts/sharepoint_list_schemas.json`.
