# Lessons Learned Scope Lock

## Closed Scope

The Lessons Learned Center is documented as a PCC-native future workstream. This package authorizes documentation updates only.

## In Scope

- Create `docs/architecture/blueprint/sp-project-control-center/future-workstreams/lessons-learned/` documentation.
- Define target architecture, system-of-record posture, developer contracts, source mapping, state machine, permission/redaction model, HBI guardrails, metrics, and fixture scenarios.
- Update governing PCC architecture docs only where necessary to recognize the module and its system-of-record posture.
- Preserve workbook fields as source mapping and seed taxonomy.

## Out of Scope

- TypeScript source code changes.
- Backend route changes.
- SPFx surface changes.
- Runtime HBI integration.
- External source-system calls.
- Evidence file upload/sync/storage.
- Phase 3 MVP promotion unless repo truth already proves it.
- Production rollout.

## Closed Decision

The Lessons Learned Center is not a closeout form. It is a lifecycle knowledge system that captures, reviews, publishes, redacts, and applies lessons across PCC.
