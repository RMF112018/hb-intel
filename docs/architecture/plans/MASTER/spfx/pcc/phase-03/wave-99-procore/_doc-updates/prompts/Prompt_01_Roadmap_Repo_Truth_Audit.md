# Prompt 01 Roadmap Repo Truth Audit

## Objective

Audit current repo truth for Phase 3 / Wave 13, identify existing Procore foundation docs, and produce a no-edit audit note before making changes.

## Required Guardrails

- Documentation/JSON only.
- No runtime code.
- No Procore SDK or dependency changes.
- No Procore live calls.
- No write-back or mirror.
- No package/lockfile mutation.

## Required Closeout

Record files changed, validation commands/results, lockfile MD5 before/after, and residual risks.
