## Suggested user-facing error copy
| Failure | Copy |
|---|---|
| invalid title | Enter a link title between 1 and 80 characters. |
| invalid URL | Enter a secure HTTPS link. |
| denied project access | This project is not available for link creation. |
| owner required | Only the person who added this link can change it. |
| link cap reached | This project already has the maximum number of active added resources. |
| source unavailable | Added project resources are temporarily unavailable. Try again shortly. |
