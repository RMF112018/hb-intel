# 4. Backend Code Must Now Consume the Trust

The existence of this federated credential does not by itself change runtime behavior.

The backend must perform the token-exchange sequence:

```text
ManagedIdentityCredential(clientId: AZURE_CLIENT_ID)
→ getToken(api://AzureADTokenExchange/.default)
→ ClientAssertionCredential(
     tenantId: AZURE_TENANT_ID,
     clientId: HB SharePoint Creator app/client ID,
     assertion callback: UAMI assertion token
   )
→ getToken(https://graph.microsoft.com/.default)
```

Only after that code change will Graph calls run under the HB SharePoint Creator application principal.
