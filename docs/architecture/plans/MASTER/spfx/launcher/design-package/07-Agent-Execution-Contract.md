# 07 — Agent Execution Contract

## Mission

Implement the launcher as a **totally fresh build**.

## Absolute rules

1. Do not copy launcher row or drawer code from the current implementation.
2. Do not preserve legacy overflow/grouping structure.
3. Do not preserve old row heading/count/container furniture.
4. Do not preserve old `More Tools` geometry drift.
5. Do not preserve old drawer helper text.
6. Do not introduce any reference in the design work to a specific SharePoint site host.

## What may be reused

Only these areas are permitted for reuse consideration:
- icon assets
- approved icon registry logic
- data adapter seams where useful
- integration seam into the homepage wrapper

## Required outcome

The final implementation must prove:
- peer-tile `More Tools`
- same tile shape across the row
- same tile language in the drawer
- one-row drawer carousel
- proper spacing
- hidden scrollbar
- clean homepage cutover

## Proof of closure

The agent must provide:
- files created
- files retired
- cutover seam used
- hosted screenshots across representative breakpoints
- verification that no site-specific hosting language was introduced into the launcher design package
