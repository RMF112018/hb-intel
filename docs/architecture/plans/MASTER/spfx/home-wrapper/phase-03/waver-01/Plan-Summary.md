# Plan Summary — Wave 01

## Objective
Move the homepage shell from “credible post-hero shell” to “governed flagship shell foundation.”

## Why this wave exists
The shell already has schemas, presets, slot metadata, and breakpoint policy. The remaining issue is that those structures are not yet decisive enough to govern the full homepage entry sequence or a future control-panel path.

## Required outcomes
1. entry-stack contract introduced and enforced
2. practical wide-state support corrected
3. Project Portfolio Spotlight promoted to first-lane eligibility
4. capability metadata materially used by layout resolution
5. persisted shell-layout groundwork established

## Out of scope
- full maintainer UI
- People & Culture rebuild
- broad unrelated visual restyling
