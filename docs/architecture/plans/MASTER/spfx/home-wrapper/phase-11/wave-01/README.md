# HB Homepage Layout Wave 01

## Goal
Replace the current shell composition model with a deterministic three-row post-launcher layout system that can express the requested sequence, handedness, and large/small hierarchy.

## Why this wave exists
The current homepage shell is not just ordered incorrectly. Its preset, band semantics, handedness model, and legality rules prevent the requested layout from being encoded cleanly.
