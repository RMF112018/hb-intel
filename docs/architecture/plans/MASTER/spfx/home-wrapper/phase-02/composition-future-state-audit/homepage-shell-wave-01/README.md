# Homepage Shell Wave 01

This prompt package is for the structural shell groundwork required before any tenant-maintainer control panel should exist.

## Wave objective
Transform `hbHomepage` from a fixed stacked host into a governed orchestration shell with:
- declarative bands / slots
- module capability metadata
- breakpoint-aware layout resolution
- protected quality guardrails
- shell-level degraded-state behavior

## Included prompts
1. `Prompt-01-Establish-Homepage-Shell-Schema-and-Slot-Registry.md`
2. `Prompt-02-Add-Module-Capability-Contracts-and-Footprint-Rules.md`
3. `Prompt-03-Refactor-HbHomepageShell-to-Resolve-From-Governed-Layout-Data.md`
4. `Prompt-04-Implement-Shell-Level-Breakpoint-Orchestration-and-Fallbacks.md`
5. `Prompt-05-Add-Shell-State-Validation-and-Hosted-Proof.md`
