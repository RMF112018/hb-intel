# Scope Boundary Notes

- a backend remediation package
- a publish-orchestrator rewrite

- a repo-wide design-system migration
- a future-wave idea dump

A Wave 01 **structural redesign closure package** for the Publisher UI seams that still materially block adoption quality.

## Boundary-call rule

If you discover adjacent work, ask:

> Is this required to close a current Wave 01 product seam in a truthful, non-hand-wavy way?

- If yes, do it now and document why.
- If no, keep it out.
