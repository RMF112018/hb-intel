# 08-Recommended-Implementation-Waves

## Wave 1 — Compact-mode and handheld redesign
Close the hardest failure first:
- featured card mobile geometry
- badge/header/avatar layout
- compact metadata and celebrate posture
- real phone/tablet-specific behavior

## Wave 2 — Information hierarchy and homepage-value rebalancing
Once compact behavior is fixed:
- reinforce the second tier below featured
- improve recent-empty and light-data storytelling
- improve archive/feed positioning relative to the primary story

## Wave 3 — Proof, accessibility, and closure hardening
After the redesign lands:
- add screenshot proof expectations
- add responsive regression guards
- confirm target sizes, focus behavior, and packaged/hosted parity
