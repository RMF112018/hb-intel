# PCC 100-Point UI/UX Evidence Taxonomy

**Document Status:** Canonical evidence taxonomy  
**Companion Scorecard:** `docs/reference/spfx-surfaces/project-control-center/PCC_100_Point_UIUX_Mold_Breaker_Scorecard.md`  
**Companion Use Guide:** `docs/reference/spfx-surfaces/project-control-center/PCC_100_Point_UIUX_Scorecard_Use_Guide.md`

## 1. Purpose

This document reconciles the full PCC scorecard evidence checklist with the current live Playwright-supported evidence subset.

The canonical evidence model includes `EV-001` through `EV-134`.

The current Playwright suite supports 80 evidence IDs:

```text
EV-037..EV-106
EV-125..EV-134
```

## 2. Collection Modes

| Mode | Meaning |
|---|---|
| Playwright-supported | Evidence is currently represented in the typed live Playwright evidence registry or output lanes. |
| Manual / documentary | Evidence is collected through repo review, audit notes, architecture docs, score rationale, or operator-supplied material. |
| Hybrid | Evidence is partially generated by Playwright but requires expert/manual completion. |
| Future candidate | Evidence could be added to Playwright later but is not currently required there. |
