# UI Doctrine breakpoint update

This package contains updated doctrine drafts that add explicit breakpoint guidance at both the shell and application level.

Files:
- `UI-Doctrine-SPFx-Governing-Standard.md`
- `UI-Doctrine-SPFx-Homepage-Overlay.md`

Primary additions:
- practical-usable-space doctrine
- mandatory shell-level breakpoint governance
- mandatory application-level breakpoint governance
- homepage entry-stack breakpoint matrix
- hero-height and action-visibility rules by breakpoint
- first-shell-lane visibility and conditional multi-column rules
- requirement that future development optimize for an exceptionally dynamic experience across multiple display conditions
