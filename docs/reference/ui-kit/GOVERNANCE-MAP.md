## Doctrine Precedence Rule

Layer 1 runtime doctrine always wins over standards, patterns, and component/layout/pattern/lane reference docs when conflict exists.

Standards operationalize doctrine and patterns guide composition. Neither standards nor patterns override doctrine authority.

Brand governance docs support source-of-truth and usage policy. They do not replace runtime doctrine authority.
