# PCC List File Format Conformance

Reformatted files: `110`

Each `pcc/lists/*.md` file now follows the same section order and field matrix structure as the supplied `Projects` reference file:

1. Objective
2. List-Level Metadata
3. Field Schema
4. Content Types / Forms / Behavioral Context
5. Relationship Observations
6. Implementation-Relevant Findings
7. Open Questions / Follow-Up Checks

Unavailable tenant-extracted values are intentionally blank for later input.
